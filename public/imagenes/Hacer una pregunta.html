    Usa algunos estilos de texto para que tu publicación sea más legible

    Saltos de línea: doble espacio o <br />al final de cada línea
    Cursiva: * asteriscos * o _computadora_
    Bold: **double asterisks** or __double underscores__

    Use 4 or 8 spaces to create nested lists

    1. List item

        * Item B, eight spaces

    Add > to the beginning of each line to create a blockquote

    > A standard blockquote is indented

    Create code fences by placing your code between sets of 3 backticks ` or use CTRL + K

    Cómo etiquetar

    Las etiquetas ayudan a las personas adecuadas a encontrar y responder su pregunta.

    • Identifique sus etiquetas completando la oración "Mi pregunta es sobre ..."
    • Incluya etiquetas que sean cruciales solo para su pregunta, como
      C#
    • Solo incluya números de versión, como
      c # -4.0
      , cuando sea absolutamente necesario
    • Utilice etiquetas populares existentes
    Si no puede encontrar una etiqueta:
    Cree nuevas etiquetas o publique sin ellas y pida a la comunidad que cree una para usted.

    Paso 1: redacta tu pregunta

    La comunidad está aquí para ayudarlo con problemas específicos de codificación, algoritmo o lenguaje.

    Evite hacer preguntas basadas en opiniones.

      • Incluya detalles sobre su objetivo

      • Describir los resultados esperados y reales.

      • Incluya cualquier mensaje de error

    2. Muestre lo que ha probado y díganos lo que encontró (en este sitio o en otro lugar) y por qué no satisfizo sus necesidades. Puede obtener mejores respuestas cuando realiza una investigación.
    3. Cuando sea apropiado, comparta la cantidad mínima de código que otros necesitan para reproducir su problema (también llamado ejemplo mínimo reproducible )
